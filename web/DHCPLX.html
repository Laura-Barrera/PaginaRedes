SERVIDOR DHCP

El Protocolo de Configuración Dinámica de Host se utiliza para asignar de forma dinámica direcciones IP y otros parámetros de configuración de red a dispositivos conectados a una red.

En un servidor DHCP, se pueden configurar los rangos de direcciones IP disponibles y otros parámetros como la puerta de enlace predeterminada, el servidor DNS y la máscara de subred. Los dispositivos cliente envían una solicitud DHCP al servidor para obtener una dirección IP y otros detalles de configuración, el servidor DHCP luego asigna una dirección IP disponible y otros detalles de configuración al dispositivo cliente.


Entrar a la terminal de linux y logearse como super usuario con el comando su -l. Luego con el comando apt-get install isc-dhcp-server se desargan los paquetes y archivos necesarios para configurar el DHCP

Acceder al archivo /etc/dhcp/dhcpd.conf para configurar el nombre del dominio y la IP del servidor y configurar el tiempo de conexion. Además se debe tener como linea sin comentar la de authoritative.


Dentro del mismo archivo /etc/dhcp/dhcpd.conf, se configura el rango de direcciones a asignar para la IP estática del servidor.

Acceder al archivo /etc/default/isc-dhcp-server y en este configurar el nombre del adaptador dentro de la interfas de IPV4, asociando asi la IP y los rangos de direcciones.


Finalmente, restablecer el servidor DHCP con el comando systemctl restart isc-dhcp-server y verificar que su estado sea activo.
