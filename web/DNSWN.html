SERVIDOR DNS

Es un servicio fundamental en la infraestructura de redes y en la comunicación entre dispositivos conectados a Internet. En Windows Server, el servicio DNS permite asignar nombres de dominio a direcciones IP, lo que permite a los usuarios acceder a los recursos de red utilizando nombres de fácil recordación en lugar de tener que recordar las direcciones IP.
En este caso, como inicialmente se instalo el servicio de AD, el servidor DNS se instalo de manera automatica y por lo tanto solo es necesario configurarlo.


Entrar al servidor DNS desde el administrador del servidor. Luego dar clic derecho sobre el registro del servidor y dar clic en Administrador de DNS.

Entrar a la zona de busqueda directa que tenga el nombre del dominio del servidor, dar clic derecho y luego dar clic en Host nuevo .


En la ventana de Host nuevo, colocar como nombre el nombre del servidor global de Windows server. Además, en la dirección IP colocar la dirección del servidor de Windows Server.

Ahora, dar clic derecho sobre la carpeta de Zona de búsqueda inversa y dar clic en Zona nueva. Esto se crea para que el DNS pueda asociar la dirección IP con e dominio del servidor


Seleccionar que la zona funcione para todos los servidores DNS que tengan relación con el dominio principal y dar clic en siguiente.

Seleccionar Zona de búsqueda inversa para IPV4 dado que la direccion estatica del servidor Windows Server esta para este mismo protocolo y dar clic en siguiente


Escribir los 3 primeros valores de la dirección IP estatica en el campo de Id. de red y dar clic en siguiente.

Seleccionar No admitir actualizaciones dinamicas dado que al tener una IP estática para el servidor DNS no se debe actualizar su información de manera constante. Luego dar clic en siguiente.


Dar clic en finalizar para completar la creacion de la zona inversa.

Luego, entrar a la carpeta creada, dar clic derecho y seleccionar Nuevo puntero (PTR).


Seleccionar el host creado en la zona directa con anterioridad para que sea el puntero de la zona inversa.

Finalmente, cuando ya se tenga el nombre del host correcto, verificar que la direccion IP del host que asocio el sistema sea correcta y dar clic en aceptar.
