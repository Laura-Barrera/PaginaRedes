SERVIDOR FTP

FTP es un protocolo de red que se utiliza para transferir archivos de un host a otro en una red TCP/IP. El servidor FTP de Debian 11 Linux se puede configurar para proporcionar diferentes niveles de acceso y permisos a los usuarios, lo que permite controlar quién puede acceder a qué archivos y qué operaciones pueden realizar.

Se puede usar para compartir archivos entre diferentes usuarios en una red local, publicar archivos en un sitio web, como imágenes, archivos de audio o video, o documentos en formato PDF, realizar copias de seguridad de archivos y almacenarlos en un servidor remoto o en la nube y descargar actualizaciones de software desde un servidor FTP público.


Entrar a la terminal de linux y logearse como super usuario con el comando su -l. Luego con el comando apt-get install vsftpd se desargan los paquetes y archivos necesarios para configurar el FTPs

Crear una carpeta en el directorio raiz del super usuario, en la cual se compartirán los archivos e información con otros usuarios. Luego se le deben dar permisos de read, write and execution a la carpeta para que cumpla con su proposito.


Acceder al archivo /etc/vsftpd.conf y validar que todas las lineas que se muestran sin comentar coincidan segun la configuración del sistema.

Validar que todas las lineas que se muestran sin comentar coincidan segun la configuración del sistema.


Validar que la linea chroot salga descomentada y coincida su valor.

Para crear los usuarios de acceso al FTP, desde el super usuario con el comando adduser seguido del nombre del usuario se podra crear el usuario con acceso a la carpeta.


Con el comando cat /etc/passwd | cut -d: -f1 se podran visualizar todos los usuarios del servidor y validar si su creacion es correcta.

Finalmente, restablecer el servidor FTP con el comando systemctl restart vsftpd y verificar que su estado sea activo.
