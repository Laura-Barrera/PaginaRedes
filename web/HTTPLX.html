PROTOCOLO HTTP

Es un protocolo de comunicación utilizado para la transferencia de datos a través de la World Wide Web.

Para Linux server es necesario implementar un servidor web externo al sistema operativo para permitir la publicación de paginas o webs relacionadas directamente con la IP y el dominio del servidor. En este caso se hizo uso de Apache, un servidor web de código abierto utilizado para alojar sitios web en Internet.


Entrar a la terminal de linux y logearse como super usuario con el comando su -l. Luego con el comando apt-get install apache2 se desargan los paquetes y archivos necesarios para configurar el servidor HTTP

Automáticamente, se relacionara el dominio e IP del Linux server con la página por defecto de Apache


Para cambiar la pagina por defecto de Apache se deben crear host virtuales en la ruta /var/wwww/. Primero se debe crear un directorio principar para alojar el nuevo sitio web y darle los permisos necesasarios.

Ahora se debe crear un archivo de configuración para relacionar el host virtual con el dominio asignado al Linux server con el comando nano /etc/apache2/sites-avalaible/example.conf y alli se configuran las lineas necesarias segun la imagen.


Ahora se debe habilitar el nuevo host virtual con el comando a2ensite example.conf y deshabilitar el sitio por defecto del apache con el comando a2dissite 000-default.conf.

Finalmente se restablece el servidor web apache con el comando systemctl restart apache2


Cuando se publique el nuevo host virtual y al acceder a la IP y el dominio del Linux server se podra visualizar en este caso la página realizada por el estudiante.
