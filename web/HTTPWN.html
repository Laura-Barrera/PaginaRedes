PROTOCOLO HTTP

Es un protocolo de comunicación utilizado para la transferencia de datos a través de la World Wide Web. Windows Server es un sistema operativo de servidor desarrollado por Microsoft y tiene una característica integrada llamada Servicios de Internet (Internet Information Services, IIS), que incluye un servidor web HTTP.
En este caso, se hace uso del HTTP para la visualización de una pagina web del tutorial de la instalación de todos los servicios y servidores del Taller en Windows Server.


Desde el Administrador del servidor dar clic en herramientas y seleccionar IIS

Dar clic derecho sobre el sitio por default, luego clic en Administrar sitio web y luego dar clic en Configuración avanzada.


Cambiar la ruta del directorio para la visualización de la página web creada.

Desde el dominio verificar la correcta visualización de la página.
