SERVIDOR SMTP

Es un protocolo de red utilizado para enviar correos electrónicos a través de la red. En Windows Server, el servicio SMTP está disponible a través del IIS (Internet Information Services), que permite a los usuarios enviar correos electrónicos a través de un servidor SMTP o tambien se puede utilizar haciendo uso de aplicaciones externas.
En este caso se hizo uso desde el Windows Server de VisendoSMTP, una aplicación de software basada en el correo electrónico que pretende sustituir a un servidor POP3 ausente que está asociado a sistemas operativos específicos. También actúa como un "puente" que puede adaptarse a los protocolos normales de correo electrónico. Por otro lado, también se utilizo desde el cliente Foxmail, un Foxmail un práctico gestor de correo, una alternativa a Outlook fácil de usar y que ofrece las funciones habituales para enviar y recibir mensajes, guardar los datos de tus contactos y configurar alarmas.


En el programa Administrador del servidor dar clic en Administrar y posteriormente en Agregar roles y caracteristicas para poder agregar el servidor SMTP a Windows Server

En el apartado de Roles del servidor seleccionar el Servidor SMTP y dar clic en siguiente.


Luego de seleccionar el servidor SMTP, avanzar sin modificar las caracteristicas predeterminadas hasta la ventana de Confirmar selecciones de instalación y dar clic en instalar.

Luego, desde el Administrador del servidor dar clic en herramientas y seleccionar IIS 6.0.


En el Administrador IIS 6.0 dar clic derecho sobre el servidor, luego en nuevo y seleccionar Servidor virtual SMTP.

En el asistente para el nuevo servidor virtual SMTP escribir el nombre del servidor y dar clic en siguiente.


Seleccionar la IP que corresponda al servidor principal que se va a asociar con el servidor SMTP.

Seleccionar la carpeta raiz creada por defecto por el servidor SMTP, que esta en la ruta C:\inepub\mailroot.


Escribir el dominio asociado al servidor SMTP y dar clic en finalizar. Además, se debe desactivar el servidor virtual de SMTP creado por defecto durante la instalación del servidor y solo dejar activo el que se ha creado manualmente.

Desde Windows Server descargar e instalar Visendo SMTP para gestionar el envio de correos electronicos con el protocolo POP3.


Asociar un nombre y el dominio del servidor SMTP a Visendo SMTP. Luego avanzar en la instalación sin modificar las caracteristicas predeterminadas y dar clic en instalar.

Verificar en Visendo SMTP que la direccion IP, la carpeta raiz y el puerto del programa sean correctos.


Crear los usuarios de correo especificos desde Visendo SMTP.

Luego de crear los usuarios de correo necesarios dar clic en finish para guardar los cambios.


Instalar Foxmail en el equipo del cliente.

Para probar el servidor SMTP desde Foxmail crear una sesión seleccionando Other MailBox.


Colocar los datos correspondientes a las credenciales de acceso y el dominio o IP asociadas al Windows Server y servidor SMTP.

Finalmente se tiene acceso a el correo electronico y se valida que se envien y reciban mensajes de manera correcta.
