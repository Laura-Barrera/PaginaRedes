CONFIGURACIÓN INICIAL - INSTALACIÓN WINDOWS SERVER

Para el correcto funcionamiento del Windows Server es necesario configurar inicialmente una dirección IP estática para el servidor, instalar el servicio de Active Directory (AD) e instalar el servidor de Internet Information Services (IIS)


IP estática

Es importante configurar una IP estática dentro del Windows Server para poder instalar de manera correcta todos los servicios y servidores adicionales.
Para este ejercicio se emplearon las direcciones IP entre el rango de 192.168.2.0 hasta 192.168.2.254

Entrar al centro de redes y recursos compartidos y dar clic en Ethernet.

En la ventana Estado de Ethernet dar clic en Propiedades.


En los elementos de conexion de la red buscar el Protocolo de Internet versión 4 y dar clic en propiedades.

Asignar una Dirección IP estática que se encuentre dentro del rango de direcciones disponibles para el servidor. Luego, asignar la dirección DNS principal que en este caso es la misma dirección de IP y un DNS secundario que por defecto puede ser 127.0.0.1.


Servicio Active Directory - AD

El servicio de AD se puede configurar dentro del Windows Server, dado que es un servicio esencial para la administración de redes empresariales en entornos de Windows Server, permitiendo una gestión centralizada y eficiente de usuarios, recursos y políticas de seguridad.
En este caso, se empleo para la creación del dominio del servidor y el manejo de usuarios y autenticación en el servidor FTP.

En el programa Administrador del servidor dar clic en Administrar y posteriormente en Agregar roles y caracteristicas para poder agregar el servicio de AD a Windows Server

En el apartado de Roles del servidor seleccionar Servicios de dominio de Active Directory y dar clic en siguiente.


Avanzar hasta la ventada de confirmar selecciones de instalación y dar clic en Instalar.

Cuando finalice la instalación del AD, dar clic en el texto de Promover este servidor a controlador de dominio para crear el dominio de acceso al servidor


Dentro del Asistente para configuración seleccionar Agregar un nuevo bosque y colocar el nombre del dominio raiz que se desee. Luego dar clic en siguiente.

Escribir una contraseña para restauración de servicios que contenga letras mayusculas y minusculas, al menos un numero y al menos un simbolo y dar clic en siguiente.


Modificar el dominio del NetBios o dejar el predeterminado por el sistema y dar clic en siguiente.

Las carpetas de almacenamiento ya tienen una ruta por defecto asi que no se deben configurar. Luego dar clic en siguiente


Finalmente, dar clic en instalar para asociar el dominio deseado con el servidor de Windows Server.

Para comprobar que el dominio y el AD quedaron correctamente instalados, desde una ventana de PowerShell ejecutar el comando Get-ADDomain y debe salir la informacion del dominio del servidor.


Internet Information Services - IIS

El servidor IIS se puede instalar dentro del Windows Server con el fin de desplegar y permitir el uso y administración de servicios web y de compartir archivos.
En este caso, se empleo para la verificacion del funcionamiento del protocolo HTTP y del servidor FTP solo de lectura.

En el programa Administrador del servidor dar clic en Administrar y posteriormente en Agregar roles y caracteristicas para poder agregar el servidor IIS a Windows Server

En el apartado de Roles del servidor seleccionar Servidor web (IIS) y dar clic en siguiente.


En el rol del servidor web IIS validar que las caracticas del servidor sean correctas y dar clic en siguiente.

Los roles adiciones del servidor IIS no se modifican. Luego, dar clic en Siguiente.


Confirmar que las caracteristicas del servidor ISS sean correctas y dar clic en siguiente.

Para validar si el servidor IIS quedo instalado de manera correcta, en el Administrador del servidor dar clic en Administrador de Internet Information Services (IIS).


La ventana del IIS que se deberá visualizar será esta.

Finalmente, verifique si el dominio que creo en el AD reconoce y carga correctamente la pagina principal del servidor IIS
