INSTALACIÓN LINUX SERVER

La instalación de Linux Server en su distribución Debian 11 implica la realización de algunos pasos y configuraciones especificas que se realizan a través de una interfaz gráfica de usuario o mediante el uso de comandos en la línea de comandos.
Es importante tener instalado un programa de máquina virtual en el equipo donde se vaya a instalar el Linux Server y configurar el tipo de conexion de red como Bridge.


Entrar a la página oficial Debian y dar clic en Descargar una imagen de instalación Es importante resaltar que Debian es una de las alternativas para descargar un servidor Linux.

Seleccionar la distibución de la imagen que desea instalar, para este caso se descargo la imagen amd64.


Cargar el archivo de Linux server en una maquina virtual y configurar las caracteristicas propias de la máquina. Seleccionar la versión de Debian que se desea instalar y dar clic o enter.

Seleccionar el idioma del sistema que se va a instalar, para este caso Español y dar clic en Continue.


Seleccionar la ubicación geográfica del equipo donde se va a instalar el servidor para relacionarlo con el tipo de teclado del equipo. Luego dar clic en Continuar.

Validar que la configuración del teclado sea correcta y dar clic en Continuar.


Asignar un nombre a la máquina o servidor y dar clic en Continuar.

Asignar un dominio al servidor de Linux como se observa en la imagen y dar clic en Continuar.


Asignar una clave al super usuario del sistema y dar clic en Continuar.

Asignar un nombre al usuario principal con el que se va a configurar el servidor y dar clic en Continuar.


Asignar una clave al usuario creado y dar clic en Continuar.

Seleccionar el metodo de partición guiado en todo el disco y dar clic en Continuar.


Seleccionar el disco local que se va a particionar para la instalación del servidor Linux y dar clic en Continuar.

Seleccionar agregar todos lod ficheros del sistema operativo en una sola partición y dar clic en Continuar.


Finalizar el proceso de particionado de discos y dar clic en Continuar.

Dar clic en escribir los cambios sobre los discos y esperar a que se realice el proceso.


Seleccionar no usar medios de instalación adicionales.

Elegir el país de replica para el gestor de paquetes, en este caso se seleccionó Estados Unidos y dar clic en Continuar.


Seleccionar el enlace de replica deb.debian.org y dar cic en Continuar.

No es necesario asignar la información de proxy para este caso, luego dar clic en Continuar.


Permitir la descarga del GRUB o cargador de arranque del sistema operativo.

Elegir el dispositov para instalar el GRUB, dar clic en Continuar y esperar a que finalice por completo la instalación.
