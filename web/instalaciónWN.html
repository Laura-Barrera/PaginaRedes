INSTALACIÓN WINDOWS SERVER

La instalación de Windows Server implica una serie de pasos que se realizan a través de una interfaz gráfica de usuario o mediante el uso de comandos en la línea de comandos.
Es importante tener instalado un programa de máquina virtual en el equipo donde se vaya a instalar el Windows Server y configurar el tipo de conexion de red como Bridge.


Entrar a la página oficial de Windows Server y dar clic en Descargar la ISO.

Completar el formulario de la página para descargar la ISO de Windows Server 2012 R2.


Seleccionar la versión de la ISO de Windows Server que necesite descargar y esperar que se descargue el archivo.

Cargar el archivo de windows server en una maquina virtual y configurar el idioma y formato del equipo de trabajo. Luego dar clic en siguiente.


Despues de seleccionar las caracteristicas de idioma dar clic en instalar ahora.

Seleccionar la version de Windows Server que se desea instarlar. Para este caso selecciono la versión estandar con interfaz gráfica.


Aceptar los términos de la instalación y dar clic en siguiente.

La instalación en este caso se realiza como personalizada, dado que se esta instalando dentro de una máquina virtual.


Esperar a que se realice la instalación completa del Windows Server 2012 R2

Asignar una contraseña al administrador del sistema.
